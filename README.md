# Airbnb Data Analysis with R

This project analyzes Airbnb listing data using R to uncover pricing patterns, host performance, and availability trends across neighborhoods. The analysis explores key factors influencing listing prices and customer experience by examining metrics such as average price by neighborhood, most expensive locations, review scores versus price, and listing availability throughout the year.

Using data analysis and visualization techniques in R, the project identifies popular neighborhoods, high-performing hosts, and relationships between pricing and customer ratings. The analysis also highlights how availability and demand patterns vary across locations.

The goal of this project is to demonstrate how data analytics can provide insights into the short-term rental market and help hosts, travelers, and businesses make informed decisions. This project showcases skills in data cleaning, exploratory data analysis, statistical analysis, and data visualization using R.
